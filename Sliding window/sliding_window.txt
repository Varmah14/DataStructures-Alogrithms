Lesson 1 — Sliding Window (fixed & variable)
When to use (spot it in <5s)

The task is about a contiguous subarray/substring.

Wording like longest / shortest / number of substrings or subarrays that satisfy some property.

If the property depends on counts/frequency and all numbers are non-negative → sliding window is ideal.
If sums include negatives, prefer prefix-sum + hashmap (next lesson).

Core idea

Grow the window on the right; when the constraint is violated, shrink from the left until valid again. Track the best answer along the way.
